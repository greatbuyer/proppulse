1. Executive Summary

PropPulse is a comprehensive US real estate analytics platform that provides interactive visualizations, market comparisons, and pricing trend analysis for residential and commercial properties across 15 US states. The platform is built with modern web technologies and is fully deployed to production with a custom domain.

Key Achievements

  • ✅ Full-stack web application built and deployed in a single session
  • ✅ Interactive US heatmap with state-level pricing data
  • ✅ Real-time data powered by Supabase (PostgreSQL database)
  • ✅ User authentication system (signup/login)
  • ✅ Data export capabilities (CSV and PDF)
  • ✅ Marketing landing page with professional design
  • ✅ Deployed to production with custom domain (berrypickle.com)

Data Coverage

The platform tracks 15 US states with 7 years of historical data (2020–2026): California, Washington, New York, Texas, Florida, Colorado, Arizona, Nevada, Ohio, Pennsylvania, Illinois, Georgia, North Carolina, Tennessee, and Michigan.

4. Features & Functionality

4.1 Dashboard Features

📊 KPI Cards

Four real-time metric cards displayed at the top of the dashboard: Median Home Price, Price Per Square Foot, Average Days on Market, and Total Active Inventory. Each card shows year-over-year change with green/red indicators.

🗺️ Interactive US Heatmap

A color-coded SVG map of the United States showing state-level data. Users can toggle between "Median Price" and "Year-over-Year Growth" views. Hovering over a state reveals a tooltip with detailed pricing data.

📈 Trend Charts

Line and area charts built with Recharts showing national pricing trends over 7 years of historical data. Supports both residential and commercial property types.

⚖️ Market Comparison

Side-by-side comparison tool allowing users to select any two states and compare metrics including median price, price per sqft, days on market, inventory, and year-over-year growth. Winner highlighting shows which market performs better in each category.

🔍 Smart Search

Instant autocomplete search component that finds any tracked state in milliseconds. Used in both the Compare View and Saved Markets features.

📋 State Rankings

Sortable table ranking all 15 tracked states by various metrics. Users can sort by price, growth rate, inventory, and other factors.

4.2 User Account Features

🔐 Authentication (Supabase Auth)

Full signup/login system with email and password. The AuthModal component supports both modes with form validation, error handling, and a polished glassmorphism UI design.

⭐ Saved Markets

Logged-in users can save favorite markets to track. The SavedMarkets panel allows adding markets via search, removing them, and toggling price alerts on/off for each saved market.

📥 Data Export

Export functionality supporting CSV (spreadsheet format) and PDF (print-ready report). The ExportButton component provides a dropdown menu accessible from the page header.

4.3 Legal & Compliance

⚖️ Disclaimer

A comprehensive legal disclaimer modal accessible from the footer covering: not financial advice, not a licensed real estate agent/broker, no guarantees of accuracy, use at own risk, and third-party data sourcing. Also featured prominently in the landing page footer.

4.4 Landing Page (Netlify)

A standalone marketing page at berrypickle.com with animated gradient backgrounds, a hero section with CTA buttons, a features grid showcasing 6 platform capabilities, a call-to-action section, and a footer with the legal disclaimer. All "Launch Dashboard" buttons link to app.berrypickle.com.
